1. 生成式AI的技術介紹與應用案例

生成式AI（如MidJourney與DALL-E）以其語言-圖像生成能力，能快速模擬可降解材料的應用場景，助力科技與日常生活的連結。

技術特性：

  • 語義理解與視覺呈現： 基於輸入描述生成具象化場景，展現材料的應用價值。
  • 高效內容創建： 快速生成多樣化設計方案，縮短產品推廣周期。

應用案例：

  • 食品包裝模擬： 基於PLA設計的環保食品包裝，展示其輕便、耐用與降解特性。
  • 農業薄膜應用： 使用PHA製成的農業覆膜，在土壤中逐步降解，減少塑膠廢棄物的累積。
  • 一次性餐具設計： 高美觀度、高功能性的餐具設計，結合耐用性與生物降解性。

2. ChatGPT-4.0：可降解性能數據的模擬與分析

生成式AI不僅支持視覺化，也能輔助材料科學中的數據建模與性能模擬。我們基於ChatGPT-4.0，開發了一套數據模擬系統，探索不同環境與溫度對降解效率的影響。

模擬程式：

import pandas as pd
import numpy as np

environments = ['Soil', 'Water', 'Air']
materials = ['PLA', 'PHA', 'Biodegradable Plastic']

data = []
for env in environments:
    for material in materials:
        for temp in range(10, 41, 10):  # 模擬10°C至40°C
            degradation_time = np.random.uniform(1, 12) * (41 - temp) / 10
            data.append([env, material, temp, round(degradation_time, 2)])

df = pd.DataFrame(data, columns=['Environment', 'Material', 'Temperature (°C)', 'Degradation Time (months)'])
print(df)

結果展示：

數據表明，材料在不同環境中的降解時間顯著不同，例如土壤中由於微生物活性高，降解速率最快。

可視化圖表：

import matplotlib.pyplot as plt
import seaborn as sns

plt.figure(figsize=(10, 6))
sns.barplot(data=df, x='Environment', y='Degradation Time (months)', hue='Material')
plt.title('Degradation Time of Materials in Different Environments', fontsize=14)
plt.xlabel('Environment', fontsize=12)
plt.ylabel('Degradation Time (months)', fontsize=12)
plt.legend(title='Material', fontsize=10)
plt.show()

3. Avogadro：分子結構建模與降解機制分析

Avogadro 是專業的化學建模工具，用於創建和優化分子結構。我們使用 Avogadro 模擬了 PLA 和 PHA 的分子結構，並通過結構分析解釋其可降解特性。

PLA 的結構特徵：

  • 含有酯基（-COO-），易受水解酶作用降解為乳酸分子。
  • 降解過程快速且環境影響小，適合用於短期使用的包裝材料。

PHA 的結構特徵：

  • 生物合成的聚合物，具有良好的生物相容性。
  • 分解產物為水與二氧化碳，適用於農業與醫療領域。
