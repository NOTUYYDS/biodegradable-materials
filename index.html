2. ChatGPT與Python：可降解性能數據的模擬與分析

生成式AI不僅支持視覺化，也能輔助材料科學中的數據建模與性能模擬。我們使用大家最熟悉不過的AI-ChatGPT，開發了一套Python的數據模擬系統，探索不同環境與溫度對降解效率的影響。

模擬程式：


import pandas as pd
import numpy as np

environments = ['Soil', 'Water', 'Air']
materials = ['PLA', 'PHA', 'Biodegradable Plastic']

data = []
for env in environments:
    for material in materials:
        for temp in range(10, 41, 10):  # 模擬10°C至40°C
            degradation_time = np.random.uniform(1, 12) * (41 - temp) / 10
            data.append([env, material, temp, round(degradation_time, 2)])
            
df = pd.DataFrame(data, columns=['Environment', 'Material', 'Temperature (°C)', 'Degradation Time (weeks)'])
df

  • 比較不同材料的環境友好性。
  • 探索溫度對降解速率的影響。

3. Avogadro：分子結構建模與降解機制分析

Avogadro 是專業的化學建模工具，用於創建和優化分子結構。我們使用 Avogadro 模擬了 PLA 和 PHA 等分子結構，並通過結構分析解釋其可降解特性。

可降解聚合物的結構與應用

可降解聚合物的結構特徵

PLA 的結構特徵

  • 全名：聚乳酸（Polylactic Acid）。
  • 結構單元：由乳酸分子（C3H6O3）聚合形成，含有酯基（-COO-）。
  • 具有良好的生物降解性，可在酶和水的作用下降解為乳酸分子。
  • 常用於短期使用的包裝材料，如食品容器和一次性用品。

PHA 的結構特徵

  • 全名：聚羥基烷酸酯（Polyhydroxyalkanoates）。
  • 結構單元：由細菌通過生物合成形成，含有酯基（-COO-）。
  • 降解產物為水和二氧化碳，對環境友好，具有良好的生物相容性。
  • 常用於農業覆蓋材料、醫療植入物和組織工程。

PBS 的結構特徵

  • 全名：聚丁二酸丁二醇酯（Polybutylene Succinate）。
  • 結構單元：由丁二酸（-CO-(CH2)2-CO-）和 1,4-丁二醇（HO-(CH2)4-OH）通過酯鍵（-COO-）聚合形成。
  • 具有良好的生物降解性，可分解為水和二氧化碳。
  • 常用於農業覆蓋膜、生物可降解塑料袋及餐具。

PVA 的結構特徵

  • 全名：聚乙烯醇（Polyvinyl Alcohol）。
  • 結構單元：由乙酸乙烯酯（Vinyl Acetate）聚合後水解形成，主鏈為碳碳骨架，側基為羥基（-OH）。
  • 具有良好的水溶性和生物相容性，容易降解。
  • 廣泛用於膠黏劑、紡織工業、醫療領域及水溶性薄膜。
